# 函館市 居住地推薦＆空間分析システム (Hakodate Housing Recommendation GNN)

公立はこだて未来大学（または函館駅前など）への通学・通勤利便性、日常の買い物環境、および地勢（標高・坂道回避）を総合的に評価し、最適な居住エリアを推薦・視覚化するGNN（グラフニューラルネットワーク）搭載の空間分析システムです。

---

## 🚀 これまでの取り組みと進化の歩み

本プロジェクトは、データの拡充から推薦アルゴリズムの高度化まで、以下のステップを経て進化してきました。

### 1. 店舗インフラデータの拡充と全自動パイプライン化
*   **取り組み**: 500件を超える最新の店舗情報（スーパー、コンビニ、ドラッグストア、飲食店、医療機関等）を取り込み、国土地理院APIから標高データを取得するキャッシュ付き処理を開発。
*   **仕組み化**: データ抽出からOSM（OpenStreetMap）歩行路網へのcKDTreeマッピング、GNNデータセット構築、3つの分析マップ生成までを約50秒で一挙に実行するオーケストレータ `run_pipeline.py` を実装しました。

### 2. 停留所位置情報の精密クレンジング (`src/correct_stops.py`)
*   **取り組み**: 国土地理院のジオコーディング仕様により、一部のローカル停留所（新函館北斗駅や上陣川など計99箇所）が「函館市役所付近」のデフォルト座標に丸め込まれていたバグを検出。
*   **対策**: 接尾辞のクレンジング、および停留所名に含まれる町名（陣川、昭和、石川など）を抽出して正しい代表位置に自動フォールバックする検索アルゴリズムを実装し、マスタデータを完璧に修復しました。

### 3. GNN推薦モデルのトポロジー改修と決定論的モデル化
*   **物理的な距離減衰の導入**: 停留所間の直線距離（km）をエッジ重みにペナルティとして反映し、「遠方の停留所が過大評価される」問題を解決。
*   **双方向（無向）グラフ化**: 従来の「停留所 ➔ 大学」の一方通行エッジを双方向（往復）化し、大学（目的地）からアクセシビリティのシグナルが正しく周辺に逆流伝播するトポロジーを確立。
*   **ランダムブレとトラフィックバイアスの根絶**: 
    *   未学習GNNのランダム初期化を constant 重み（`1.0`）に固定し、計算の再現性を100%担保。
    *   GNNの初期特徴量を「目的地インジケーター（1次元）」に純化し、市電のような高頻度交通が「便数の多さ」だけで通学ポテンシャルを不当に独占していたトラフィックバイアスを完全に排除。停留所自体の利便性はGNNの外部で後ブレンド（20%分）するモデルに刷新しました。

## 📊 生成される分析・可視化マップ (data/ 以下)

本システムは、分析結果をビジュアル豊かな3つの高解像度プロットとして出力します。

1.  **空間分布マップ (`data/hakodate_plot.png`)**
    *   薄グレーの道路網の上に、停留所（青丸）と店舗（オレンジ四角）の分布を重ね合わせ、都市インフラの密度を俯瞰できます。
2.  **店舗カテゴリ別集積マップ (`data/hakodate_stores_plot.png`)**
    *   スーパー、コンビニ、ドラッグストアなど6カテゴリを色分けプロット。美原の商業コアや五稜郭の飲食密集が一目で理解できます。
3.  **空間クラスタリングマップ (`data/hakodate_spatial_clusters.png`)**
    *   GNNが路線のトポロジーと標高・店舗密度から導き出した8つの「都市セグメント」をカラー分類。GTFSの市電軌道（赤い実線）も重ねてプロットされています。

## 🛠 使い方と環境構築

### 1. 仮想環境の再現
プロジェクトの依存ライブラリをインストールします。
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. パイプラインの一括更新
店舗情報の更新や、停留所マスタを再マージしてすべてのグラフ・画像を再生成したい場合：
```bash
python3 run_pipeline.py
```

### 3. 居住地推薦エンジンの実行
プレフィックスパラメータを変更して、学生のライフスタイルに合わせた最適な居住エリアをランキング出力します。

*   **【推奨】王道の学生一人暮らし設定 (美原エリア1位)**
    *   通学しやすく、買い物が非常に便利で、かつ多少の坂道（美原の標高30mなど）は許容する設定。
    ```bash
    python3 src/recommend.py --target "公立はこだて未来大学" --weight_transit 1.8 --weight_life 0.8 --weight_slope 0.4
    ```
*   **【通学・コスパ最優先】大学直近設定 (赤川エリア1位)**
    *   買い物や坂は割り切り、大学までの近さだけを極限まで求める設定。
    ```bash
    python3 src/recommend.py --target "公立はこだて未来大学" --weight_transit 2.0 --weight_life 0.2 --weight_slope 0.5
    ```
*   **【街中ライフ】平地・商業重視設定 (函館駅前1位)**
    *   坂道は絶対に嫌で、外食やバイトが便利な街中に住む設定。
    ```bash
    python3 src/recommend.py --target "公立はこだて未来大学" --weight_transit 0.5 --weight_life 1.0 --weight_slope 2.0
    ```
